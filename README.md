## 🌟 About This Repository

This repository is a **complete hands-on journey of Spring AI**, where I explored and implemented:

* 🤖 LLM Integration (OpenAI / Ollama)
* 🧠 RAG (Retrieval Augmented Generation)
* 🛠️ Custom Advisors & System Prompting
* 🔗 Tool Calling & External APIs
* 💬 Chat Memory Management
* 🗂️ Real-world AI Applications

👉 This is not just theory — every concept is implemented through **real working projects**.

## 📂 Project Structure (Learning Flow)

```
SPRING-AI-FULL-SERIES/
│
├── advisor_app                → Custom advisors & system prompting
├── chatclient-api             → Understanding chat clients
├── memory_app                 → Storing chat memory in DB
├── rag_app                    → RAG implementation
├── advance_rag                → Advanced RAG + ETL pipeline
├── tool-calling-example       → External tool integration
├── help-desk-backend          → AI-powered helpdesk backend
├── helpdesk-frontend          → Frontend for helpdesk
├── mcp / mcp-app              → External API calling system
├── spring-ai-*                → Different AI configurations
└── boot-gemini-app            → Gemini AI integration
```
